Earth imagery sources

- `earth-day-blue-marble.jpg` was derived from NASA Visible Earth Blue Marble Next Generation:
  https://eoimages.gsfc.nasa.gov/images/imagerecords/74000/74393/world.topo.200407.3x5400x2700.jpg
- `earth-night-lights.jpg` was derived from NASA Visible Earth Earth at Night:
  https://eoimages.gsfc.nasa.gov/images/imagerecords/36000/36008/lights_2003_lrg.jpg

Both files were resized to 4096x2048 for safer WebGL texture uploads in Cesium.
